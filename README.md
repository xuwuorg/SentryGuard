哨岗守卫 SentryGuard
===========================
## 关于哨岗守卫
名字来源于war3的一个商店道具名称

## 关于这是干嘛的
哨岗守卫分为两部分，SentryGuard.exe是一个带UI的进程注入器。MFC.dll是用于注入到三方进程的。MFC.dll的用途在于在进程内通过特征码搜索得到MFC程序大部分的事件消息的回调并且将当前执行的回调地址通过对话框弹出来。
